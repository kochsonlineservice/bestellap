# Friseur & Kosmetikstudio Website

Eine moderne und responsive Website für ein Friseur- und Kosmetikstudio.

## Features

- Responsive Design
- Modernes und feminines Layout
- Leistungen
- Preisliste
- Bildergalerie
- Kontaktformular
- Google Maps
- Instagram-Verlinkung
- Impressum & Datenschutz

## Technologien

- HTML5
- CSS3
- JavaScript
